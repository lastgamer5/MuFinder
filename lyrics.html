Who we are

We are Campbell Highschool students participating in a music technology innovation project. This project requires for us students to think outside the box and come up with something new and creative. For our project, we decided to make a theoretical website that takes your audio file and outputs the exact song that audio is from. This is a simple website meant to help people find the song stuck in their heads. However, if you can't get the audio you can click on the "Type lyrics to audio" tab in order to type in lyrics instead of a file. I made this by using 3 programming languages used to make common day websites. These languages include HTML, CSS, and Javascript. Each has a specific function to making this site viewable and usable.

How it works

This website works by using Chatgpt's AI language model in order to detect and find the song you've been looking for. After only one year of testing, experimenting, and talking with professional software developers and data analysis persons we can finally say that MuFinder exists. The skills needed to create such a monumental task were great, but were worth it.
